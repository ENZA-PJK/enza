# Horror Movie Website

## Overview
This project is a horror movie website that provides information about various horror films, reviews, and a contact form for user inquiries. The website is designed to be visually appealing and user-friendly, catering to horror movie enthusiasts.

## Project Structure
```
horror-movie-website
├── src
│   ├── index.html          # Main homepage of the website
│   ├── pages
│   │   ├── movies.html     # Information about various horror movies
│   │   ├── reviews.html    # Reviews of horror movies
│   │   └── contact.html     # Contact form for user inquiries
│   ├── css
│   │   └── styles.css      # CSS styles for the website
│   ├── js
│   │   └── script.js       # JavaScript for interactive features
│   └── assets
│       └── fonts           # Custom font files
└── README.md               # Documentation for the project
```

## Features
- **Homepage**: A welcoming page that introduces the website and provides navigation to other sections.
- **Movies Page**: Detailed descriptions, trailers, and images of various horror movies.
- **Reviews Page**: User-submitted reviews and ratings for horror films.
- **Contact Page**: A form for users to submit inquiries or feedback.

## Setup Instructions
1. Clone the repository to your local machine.
2. Open the `index.html` file in a web browser to view the homepage.
3. Navigate through the website using the provided links.
